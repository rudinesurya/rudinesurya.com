Object Pooling

About


This Object Pooling is a robust and lightweight pooling system for unity built for optimizing games that uses alot of objects over and over again.

This asset is created using Unity3D.

Description

  • Objectives : By managing a pool of instances and recycling objects that are not in use at that point of time, we can save on CPU overhead & memory that are required to instantiate & kill the objects. So, pool manager can enhance performance when we are required to work with huge amount of objects that are needed frequently but only for a short amount of time.
  • Applications : Spawning of bullets, particle effects, decals...
