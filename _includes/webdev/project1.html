Job catalog web that uses React (Redux, Axios)

About



This is an app for quick contract job, similar to the likes of Meetup. It has some form of social networking feature, eg. the ability to view user's profile and chat.

Description

  • Adhering to best practices recommended for React. Only root components are made to be stateful. ie, they will have dependencies to 3rd party web api and be passing props down the tree to the stateless children components.
  • Uses Redux for global state management.
  • Communication with backend services provided by my node server to perform CRUD operations and which also supports authentication via JWT.
  • Experience with debugging a web application using the chrome console and redux dev tools chrome extension.
