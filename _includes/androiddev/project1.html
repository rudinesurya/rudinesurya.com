Job catalog app that runs on native android (MVVM, Retrofit, Kodein, Firebase, Maps)

About



This is an app for quick contract job, similar to the likes of Meetup. It has some form of social networking feature, eg. the ability to view user's profile and chat.

Description

  • Clean Design Concept and follows Material Design.
  • MVVM Architecture Adopted.
  • Repository layer pattern.
  • Experience with unit testing using JUnit and Mockito.
